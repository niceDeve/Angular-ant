How many states does a task have ?

A task can have four states in different periods.

Analysing -- Engine is analysing magnet url in order to get necessary torrent information. This can take a long period.

Running -- Task is running.

Stopped -- Task has been stopped.

Completed -- Task has been Completed.

What is itorrents service ?

itorrents is a free service for caching torrent files online. ANT Downloader will always try to get torrent infomation from magnet url to reduce waiting time. If itorrents has no infomation, ANT Downloader will use engine to resolve magnet url.
